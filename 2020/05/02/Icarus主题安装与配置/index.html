Icarus主题安装与配置

yilia主题共能不够丰富，换用Icarus主题

1.安装

github地址：icarus

安装完成后，在 _config.yml 配置文件中更改主题为icarus

通过hexo s 进行预览，第一次预览时会提示需要的依赖包，根据提示将安装包安装好(根据提示中的 `npm install …… ，把这一句复制粘贴运行即可)

依赖包安装好后，再次通过 hexo s 进行预览，第一次运行，会生成 _config.yml 配置文件，此时在 localhost:4000中应该就可以看到Icarus主题已经安装好了
