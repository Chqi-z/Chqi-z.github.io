搭建个人博客

1. 安装nodejs

下载nodejs.org

查看nodejs版本node -v

查看npm版本 npm -v

2. 安装hexo框架

换源npm install -g cnpm --registry=https://registry.npm.taobao.org

查看cnpm版本cnpm -v

注：powershell下无法使用

查看cnpm文件

猜测Windows对脚本支持不太好，但有.cmd文件，故尝试使用cmd

安装hexo cnpm install -g hexo-cli

验证hexo版本 hexo -v

创建blog文件夹（在家目录下创建）cd ~ mkdir blog 进入blog文件夹 cd blog

3. 开始使用hexo

初始化hexo

以管理员权限打开cmd

进入到blog文件夹

初始化blog hexo init

启动本地博客服务 hexo s (s for sever)

在localhost:4000就可以访问博客了

创建第一篇博客文章 hexo n "the first blog" (n for new)

退回blog文件夹，再清理一下bloghexo clean 再生成 blog g (g for generate)

4. 把博客部署到GitHub上

新建一个仓库

在blog文件夹下安装git部署插件 cnpm install --save hexo-deployer-git

配置 _config.yml

部署到远端 hexo d (d for deploy)

遇到以上问题

用git设置用户名和邮箱 git config --global user.name "Your Name" git config --global user.email "you@example.com"

再次使用 hexo d

仓库已经有内容了

已经可以使用Chqi-z.github.io访问了

5. 更换主题

在blog目录下 git clone https://github.com/litten/hexo-theme-yilia.git theme/yilia

theme目录下已经有yilia了

修改 _config 配置

再清理一下并生成一下 hexo clean hexo g

hexo s 并在localhost:4000下查看

浏览器查看

推到远端 hexo d

在Chqi-z.github.io下查看

已生效，如果主题没变过去就刷新试一下

emm~

感觉Windows下还是没有Linux或者macos下爽
